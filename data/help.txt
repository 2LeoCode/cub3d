The map file must be a valid .cub file

Available options:
  -h, --help		Shows help message, also ignores any other argument
			and does not start the program
  -s, --save		Saves the first rendered image at program start in
			<capture> folder
  -b, --bonus		Enables the bonus part.

A valid .cub map file must be formatted like this:
<n><s>R <s>[screen x] <s>[screen y]
<n><s>NO <s>[path to north walls texture]<n>
<n><s>SO <s>[path to south walls texture]<n>
<n><s>WE <s>[path to west walls texture]<n>
<n><s>EA <s>[path to east walls texture]<n>
<n><s>S <s>[path to object sprite texture]<n>
<n><s>F <s>[floor color R]<s>,<s>[floor color G]<s>,<s>[floor color B]
<n><s>C <s>[ceiling color R]<s>,<s>[ceiling color G]<s>,<s>[ceiling color B]
<n>[map view from top]<n>
  0:		floor
  1:		wall
  2:		entity
  N/S/W/E:	User spawn point and looking direction
The map must be surrounded by '1'

  Example:
  1111111
  1020001
  1001N01
  1001021
  1111111

<s>: arbitrary number of spaces
<n>: arbitrary number of new lines

Example: default map file: [default.cub]
R 800 600
NO ./assets/default_north_texture.xpm
SO ./assets/default_south_texture.xpm
WE ./assets/default_west_texture.xpm
EA ./assets/default_east_texture.xpm
S ./assets/default_sprite_texture.xpm
F 220,100,0
C 225,30,0

Note: Images have to be in .xpm format.
The order of the parameters at the beggining does not matter,
for example this is a valid .cub file configuration:
S ./path_to_sprite_texture
WE ./path_to_west_texture
R 800 600
NO ./path_to_north_texture
C 225,30,0
EA ./path_to_east_texture
F 220,100,0
SO ./path_to_south_texture

Return values:
  0  succes exit
 -1  failure exit
